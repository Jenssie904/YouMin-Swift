# YouMin-Swift
## 本项目使用swift开发高仿游民星空项
- 使用RxSwift项目，采用MVVM架构
- 其他库:HandyJSON,RxMoya,Kingfisher,Alamofire等
- 数据均来源于原app项目抓取
- 布局方式:
  - autolayout+xib  因为不喜欢花时间在UI上所以项目使用这种布局方式
  - SnapKit 
  - subviewsLayout+frame
