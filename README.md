# YouMin-Swift
## 本项目使用swift开发高仿游民星空项
- 使用RxSwift项目，采用MVVM架构
- 其他库:HandyJSON,RxMoya,Kingfisher,Alamofire等
- 数据均来源于原app项目抓取
- 布局采用xib+autolayout
